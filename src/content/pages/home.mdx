A content-focused, side-anchored minimalist theme designed for developers, writers, and digital gardeners.
Built on [Astro]. Nothing in the way, just your words on a clean, fast canvas.

This site is the live demo and documentation for Sidey.
Head over to [Getting Started] to set up your own.

## Features

- **Minimal by design**:
  A small, focused project with little styling and less clutter.

- **Typography first**:
  Set in [Geist] for body text and [Geist Mono] for code. The reading experience is the whole design.

- **Expressive Code**:
  Syntax highlighting powered by [Expressive Code], with line numbers and collapsible sections out of the box.

- **Flexoki color scheme**:
  A paper-like color palette using [Flexoki], designed for reading comfort.

- **One config file**:
  Everything lives in `sidey.config.ts` at the root. Fill it in once, forget about it, and start writing.

## Tech stack & Deployment

It's a static site built with [Astro] and [Tailwind CSS]. Deploy anywhere that supports static site hosting, such as [Cloudflare Pages], [GitHub Pages], or [Netlify].
